🧮 Advanced Scientific Calculator Pro

A powerful, modern calculator application built with Python that combines professional functionality with an intuitive user interface. Perfect for students, professionals, developers, and anyone who needs advanced calculation capabilities.

✨ Features
🎯 Core Functionality

Basic Operations: Addition, subtraction, multiplication, division
Scientific Functions: Trigonometric, logarithmic, exponential functions
Programming Mode: Binary, hexadecimal, octal number systems
Statistical Analysis: Mean, median, mode, standard deviation, variance
Function Graphing: Plot mathematical functions with customizable ranges
Memory Operations: Store, recall, add, and subtract from memory

🎨 Modern Interface

Dual Themes: Dark and light mode support
Tabbed Interface: Organized functionality across multiple tabs
Responsive Design: Scales beautifully across different screen sizes
Professional Styling: Modern flat design with smooth animations
Status Bar: Real-time feedback and precision display

💾 Data Management

Session Management: Save and load calculation sessions
History Tracking: Comprehensive calculation history with timestamps
Data Import/Export: Support for CSV and JSON file formats
Auto-save Settings: Automatically saves user preferences

⌨️ User Experience

Full Keyboard Support: Complete keyboard navigation and shortcuts
Precision Control: Adjustable decimal precision (1-15 digits)
Error Handling: Graceful error management with helpful messages
Multi-language Ready: Architecture supports easy localization

🚀 Quick Start
Prerequisites
bashPython 3.7 or higher
pip (Python package installer)
Installation

Clone the repository
bash
git clone https://github.com/clowno7/calculator.git
cd calculator

Install required dependencies
bash
pip install -r requirements.txt

Run the calculator
bash
python calculator.py


Alternative Installation (Standalone)
Download the latest release from Releases for a standalone executable.
📋 Requirements
Create a requirements.txt file with:
tkinter (usually included with Python)
matplotlib>=3.5.0
numpy>=1.21.0
